# Contributing to NullMuseum

Thank you for your interest in contributing! We welcome all community contributions.

## Image Contributions

See our [contribution guide](./contribute.md) for detailed instructions on adding images.

## Code Contributions

### Reporting Issues

- Use GitHub Issues
- Provide clear descriptions
- Include steps to reproduce bugs

### Pull Requests

1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Test thoroughly
5. Submit a PR with a clear description

### Development Setup

```bash
npm install
npm run dev
```

## Code of Conduct

- Be respectful and inclusive
- Provide constructive feedback
- Focus on collaboration

Thank you for helping make NullMuseum better!
